## 📖 About

**WildHaven** is a front-end wildlife conservation website built with pure HTML, CSS, and JavaScript. The site encourages visitors to support wildlife through donations, volunteering, and awareness — with a mission to *"Give Love for Saving World Animals."*

---

## ✨ Features

- 🏠 **Hero Section** — Inspiring landing section with a call-to-action donation button
- 🛡️ **Feature Highlights** — Safe Shelter, Safe Water, Ecology Save, and Environment sections
- 🐾 **About Section** — Mission, Vision, and Next Plan tabs with an interactive wildlife image gallery
- 🌍 **Services Section** — Save Nature, Save Ecology, Tree Plantation, and Clear Ocean programs
- 💚 **Donation Cards** — Fundraising cards with progress bars and goal tracking
- 🗓️ **Events & Programs** — Upcoming conservation events listing
- 💬 **Testimonials** — Community feedback section
- 🤝 **Partners Section** — 324+ trusted global partners showcase with hover effects
- 📸 **Instagram Feed** — Wildlife photo gallery strip
- 📱 **Responsive Design** — Mobile-friendly with a hamburger navigation menu
- 🌐 **Language Switch** — English / Sinhala language selector in the header

## 🚀 Getting Started

### Prerequisites

No installations required — just a modern web browser.

### Run Locally

1. **Clone the repository**
   ```bash
   git clone https://github.com/ImashaSamodee/WildHaven_Webaite.git
   ```

2. **Navigate into the project folder**
   ```bash
   cd WildHaven_Webaite
   ```

3. **Open in browser**
   - Double-click `index.html`, **or**
   - Use the **Live Server** extension in VS Code for hot reload
